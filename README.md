# 資本的支出判定コンテキスト

資本的支出と修繕費の判定を行うために必要な税法関連のコンテキスト資料集です。

## 概要

このリポジトリには、固定資産への支出が「修繕費（損金/経費）」か「資本的支出（資産計上・減価償却）」かを判定するために必要な税法資料が含まれています。

   - 7-8-6の2: ソフトウェアに係る資本的支出と修繕費

## 判定の基本原則

### 資本的支出（資産計上）となるもの
- 資産の価値を高めるもの
- 耐久性を増すもの
- 使用可能期間を延長させるもの
- 用途変更を伴う改造・改装

### 修繕費（損金算入）となるもの
- 通常の維持管理
- 原状回復
- 20万円未満の支出
- おおむね3年以内の周期で行われる修理・改良

### 形式基準
- 60万円未満
- 前期末取得価額の10%以下

### 区分特例（簡便法）
- 支出額の30%相当額と前期末取得価額の10%相当額のいずれか少ない方を修繕費、残額を資本的支出とする方法（継続適用要件あり）

## 使用方法

1. **AI判定システムへの入力**: `llm.txt` を AI のコンテキストとして読み込ませる
2. **個別判定**: `src/` フォルダ内の各ドキュメントを参照する
3. **フロー図**: `judgingflow.md` で視覚的に判定プロセスを確認する
4. **具体例**: `資産税有無判定チャート具体例.txt` で類似ケースを参照する
